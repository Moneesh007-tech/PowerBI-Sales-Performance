# Business Sales Performance Dashboard
This project was completed as part of my Data Analytics internship.

## Key Features
* **Total Revenue KPI:** Real-time tracking of sales.
* **Regional Analysis:** Insights into performance across West, East, Central, and South regions.
* **Category Performance:** Breakdown of Technology, Furniture, and Office Supplies.
* **Trend Analysis:** Visualizing sales growth over time.

## Tools Used
* Power BI Desktop
* DAX (Data Analysis Expressions)
* Excel/CSV Data Sources
